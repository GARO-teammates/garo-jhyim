# 빠른 시작 가이드

## 1단계: 의존성 설치

```bash
pip3 install numpy pyserial opencv-python
```

## 2단계: 실행

```bash
cd RRR_GUI_Package_v7
./start_gui_v7.sh
```

## 3단계: 기본 사용

### 로봇 없이 시각화만 사용

1. GUI 실행
2. **시뮬 연결** 클릭
3. 서보 슬라이더 조작 → RViz에서 확인

### IK 사용 (엔드이펙터 위치 제어)

1. **IK 활성화** 체크
2. X, Y, Z 슬라이더 조작
3. 그리퍼가 해당 방향으로 이동

### 실제 로봇 연결

1. USB 케이블로 로봇 연결
2. 시리얼 포트 선택 (보통 `/dev/ttyUSB0`)
3. **로봇 연결** 클릭
4. **기본자세** 클릭하여 초기화

## 단축키

- 슬라이더 좌측 `[최]` 버튼: 최소값
- 슬라이더 우측 `[기]` 버튼: 기본값
